# 🛡️ Promcrypt Terminal 

**Promcrypt** is a professional‑grade obfuscation suite for Lua scripts and text files. It transforms readable source code into a complex, non‑linear structure, making reverse‑engineering and tampering extremely difficult — without compromising runtime performance.

---

## ✨ About

Protect your intellectual property with **Promcrypt**. Whether you're developing for Lua 5.1, LuaU (Roblox), or any Lua‑based environment, Promcrypt gives you the tools to secure your code with multi‑layer obfuscation, smart presets, and granular control.

---

## 🚀 Key Features

- **Multi‑Layer Obfuscation**  
  Combines variable renaming, string splitting, and control flow flattening.

- **Smart Presets**  
  Choose from *Minify* (speed‑focused) to *Strong* (maximum security).

- **Custom Configuration**  
  Fine‑tune every aspect of the obfuscation process via the *Custom* panel.

- **Real‑time Terminal**  
  Monitor the obfuscation process with live logs and status updates.

- **Cross‑Platform**  
  Supports Lua 5.1 and LuaU (Roblox) environments.

---

## 📖 Usage Instructions

1. **Upload**  
   Drag & drop your `.lua` or `.txt` file into the terminal, or click the **Upload File** button.

2. **Select Preset**  
   Choose a security level. Use *Strong* for production scripts.

3. **Run**  
   Click the **Run** button to start the process. Watch the terminal for progress.

4. **Export**  
   Once complete, use **Download Result** or **Copy Result** to retrieve your protected code.

---

## 💡 Tips & Troubleshooting

| Issue | Solution |
|-------|----------|
| **Large Files** | Processing very large scripts may take a few moments. The terminal will update once complete. |
| **LuaU Support** | If obfuscating for Roblox, ensure you select a preset that supports LuaU syntax. |
| **Error Logs** | If obfuscation fails, check the terminal for red error messages. Most issues are due to syntax errors in the source file. |
